컴퓨터 공학과 소개

학과 개요

컴퓨터 공학과는 컴퓨터 하드웨어와 소프트웨어를 설계, 개발, 운영하는 분야를 다룹니다. 이론과 실습을 통해 컴퓨터 시스템의 원리와 응용 방법을 배웁니다.

교육 과정

컴퓨터 공학과에서는 다음과 같은 과목들을 배웁니다:

  • 프로그래밍 언어 (C, C++, Java, Python 등)
  • 알고리즘과 데이터 구조
  • 운영체제 (Windows, Linux 등)
  • 컴퓨터 네트워크와 보안
  • 데이터베이스 시스템
  • 소프트웨어 공학과 프로젝트 관리
  • 인공지능과 머신러닝
  • 임베디드 시스템과 IoT

또한 실습을 통해 이론을 적용하고 프로젝트를 수행합니다.

진로 전망
